# Employee Management System

A Geo-Fenced Employee Management System developed using React.js and Vite to streamline employee attendance, payroll, and workforce management operations.

## Features

* Employee Registration and Management
* GPS-Based Attendance Tracking
* Geo-Fencing for Authorized Check-In Locations
* Attendance Monitoring Dashboard
* Employee Performance Tracking
* Payroll Management
* Responsive and User-Friendly Interface
* Real-Time Workforce Monitoring

## Technologies Used

* React.js
* JavaScript (ES6+)
* Vite
* HTML5
* CSS3

## Purpose

This project was developed to help organizations efficiently manage employee records, attendance, and payroll while ensuring employees can mark attendance only from authorized locations using geo-fencing technology.
